# 🚀 DecodeLabs Landing Page

A modern, fully responsive educational landing page designed for DecodeLabs. This project showcases best practices in front-end development using semantic HTML5, modern CSS, and Vanilla JavaScript without relying on any external frameworks.

## 📌 Project Overview

The DecodeLabs Landing Page is designed to promote web development training programs through an engaging and responsive user interface. It features smooth animations, mobile-friendly navigation, interactive statistics, and a contact form with validation.

The project follows modern UI/UX principles and demonstrates responsive web design techniques.

---

## ✨ Features

- Responsive Navigation Bar
- Mobile Hamburger Menu
- Hero Section with Call-to-Action Buttons
- Feature Cards Section
- Course/Module Showcase
- Animated Statistics Counter
